# 24시간! · 청소년 정치 성향 설문조사

민족사관고등학교 주제탐구 팀 **「24시간!」** 의 1차 검사용 정치 성향 설문조사 웹사이트입니다.

## 연구 개요

- **주제** &nbsp;청소년의 정치 진영 인식과 정치 성향에 따른 언어 사용의 차이
- **연구자** &nbsp;김예준 · 송경은 · 임성후 · 임유진
- **지도교사** &nbsp;김상래
- **웹사이트 제작 및 기술 지원** &nbsp;심재민

## 설문 구성

| 영역 | 문항 수 |
| :--- | :---: |
| 사회 (Society) | 8 |
| 경제 (Economy) | 5 |
| 안보 (Security) | 4 |
| **합계** | **17** |

추가로 본인이 인식하는 정치 성향 자기 평가(5단계)와, 민사고 학생 대상 2차 실험 참가 의사 확인 문항이 포함됩니다.

## 점수 산정 방식

각 보기의 점수는 다음과 같이 부여됩니다.

| 보기 | 점수 |
| :--- | :---: |
| 진보 | 0 |
| 중도진보 | 3.3 |
| 중도보수 | 6.7 |
| 보수 | 10 |

17문항의 평균값으로 최종 성향을 판정합니다.

| 평균 구간 | 결과 |
| :--- | :--- |
| 0 ~ 2.5 | 진보 |
| 2.5 ~ 5 | 중도진보 |
| 5 ~ 7.5 | 중도보수 |
| 7.5 ~ 10 | 보수 |

## 사용 방법

`index.html` 을 브라우저에서 열면 바로 동작합니다. 외부 의존성은 폰트(CDN)뿐이며, 별도의 빌드 과정이 필요 없습니다.

```bash
open index.html
```

## Google Sheets 연동 (응답 수집)

기본 상태에서는 응답이 어디에도 저장되지 않습니다. 연구 데이터를 모으려면 아래 절차로 본인 구글 계정에 데이터를 쌓이게 할 수 있습니다.

1. 새 [Google Sheets](https://sheets.new) 파일을 만듭니다.
2. 메뉴에서 **확장 프로그램 ▸ Apps Script** 를 엽니다.
3. 기본 `Code.gs` 내용을 모두 지우고, 본 저장소의 [`apps-script/Code.gs`](./apps-script/Code.gs) 내용을 그대로 붙여넣은 뒤 저장합니다.
4. 우측 상단 **배포 ▸ 새 배포 ▸ 유형: 웹 앱** 을 선택하고,
   - **다음 사용자 인증** : 본인
   - **액세스 권한** : 모든 사용자

   로 설정한 뒤 배포합니다.
5. 표시되는 **웹 앱 URL** (`https://script.google.com/macros/s/.../exec`) 을 복사합니다.
6. `index.html` 상단의 `WEBHOOK_URL` 상수에 그 URL 을 붙여넣고 커밋합니다.

```js
const WEBHOOK_URL = "https://script.google.com/macros/s/AKfyc.../exec";
```

이후 응답자가 1차 검사를 끝내면 시트에 한 행씩 자동으로 쌓이며, 다음 컬럼이 저장됩니다.

| 컬럼 | 설명 |
| :--- | :--- |
| 제출시각 | 자동 |
| 소속학교 / 학번 | 응답자가 입력한 값 |
| Q1 ~ Q17 | 각 문항의 가중 점수 (0 / 3.3 / 6.7 / 10) |
| 사회·경제·안보 평균 | 카테고리별 평균 |
| 전체평균 / 판정카테고리 | 17문항 평균과 그에 따른 진보·중도진보·중도보수·보수 판정 |
| 본인인식 | 자기 평가 결과 |
| 2차실험참가 | Y / N (민사고 응답자에 한함) |
